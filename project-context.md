## Publishing Setup
- **Direct store**: Gumroad (MCP connected — `rmarescu/gumroad-mcp`)
- **Ebook distribution**: Wide via Draft2Digital (not KDP Select exclusive)
- **Print**: KDP Print + IngramSpark dual strategy
- **Audiobook**: ElevenLabs Studio → INaudio (wide) + KDP Virtual Voice (Amazon)
- **PDF ebooks**: HTML→PDF via Puppeteer screenshot workflow
